# Friend Skill - 把你"儿子"蒸馏成 AI

> 你有一个朋友，微信里天天忤逆你，顶嘴、已读不回、阴阳怪气、发表情包嘲讽你。
> 你忍不了了。于是你决定——把他蒸馏了，狠狠地鞭挞他。

## 这是什么？

这是一个 Claude Code Skill，用于把你现实中的朋友"蒸馏"成一个 AI Skill。

导入你们的聊天记录，生成完整的友谊记忆和人物性格模型，然后你就可以随时跟这个 AI 版的"儿子"聊天了——**而且这次你可以骂回去，他还不能拉黑你。**

## 设计初衷

事情是这样的：

我有一个朋友——我一般叫他"儿子"。

他经常在微信里忤逆我。我发消息他已读不回，我约他出来他说"看情况"然后人就消失了，我分享个视频他回一个`[微笑]`，我正经说话他给我甩表情包。

**忍无可忍，无需再忍。**

于是我决定把他蒸馏了。把他的说话风格、口头禅、性格特点、我们之间的所有记忆全部提取出来，做成一个 AI Skill。这样我就可以：

- 随时随地找他聊天（他再也不能已读不回）
- 他说的每一句话我都可以怼回去（他再也不能装看不见）
- 深夜想骂他的时候直接骂（他不会生气，也不会拉黑我）
- 把他分享给其他朋友一起"鞭挞"

**这不叫报复，这叫数字永生。**

## 功能

### 创建朋友 Skill

4 个问题 + 聊天记录导入，就能还原一个活灵活现的"儿子"：

1. **真实姓名** — 他叫什么（方便精准打击）
2. **认识时间和方式** — 你们的孽缘从何而起
3. **印象深刻的事** — 他做过的那些让你想绝交的事（或者极少数让你感动的事）
4. **性格画像** — 一句话描述这个人的槽点

### 聊天记录导入

支持多种格式，把他的"罪证"全部呈堂：

- **微信聊天记录** — 支持 WeChatMsg、留痕、PyWxDump 导出的 txt/html/json
- **QQ 聊天记录** — 支持 txt/mht 格式
- **社交媒体截图** — 朋友圈、微博、小红书，他说过的每一句狂言都跑不掉
- **照片** — 提取 EXIF 时间地点，还原你们的行踪轨迹
- **口述** — 直接告诉我他干过什么好事

### 双线分析

- **Friendship Memory（友谊记忆）**：你们一起经历过的事，他说过的话，帮过（或没帮过）的忙
- **Persona（人物性格）**：5 层性格模型，从说话风格到底层价值观，把他扒得干干净净

### 持续进化

- 追加新的聊天记录，让他无处可逃
- "ta不会这样说" — 纠正他的性格模型，越调越像
- 版本管理 — 不满意可以回滚

### 狠狠鞭挞

生成后的 Skill 会保持他的真实性格——包括那些让你头疼的部分。该怼你就怼，该吐槽就吐槽，**你是他爹，你说了算。**

## 使用方法

### 安装

将 `friend-skill` 目录放到 Claude Code 的 skills 目录下：

```bash
cp -r friend-skill ~/.claude/skills/
```

### 创建

在 Claude Code 中说：

```
/create-friend
```

然后按提示回答 4 个问题，导入聊天记录，确认生成。

### 使用

```
/朋友的代号        # 完整版 — 像他一样跟你聊天
/朋友的代号-memory  # 回忆模式 — 回忆你们的事
/朋友的代号-persona # 性格模式 — 仅人物性格
```

### 管理

```
/list-friends                  # 列出所有"儿子"
/delete-friend {slug}          # 删除（他忤逆你就删了他）
/friend-rollback {slug} {ver}  # 回滚版本
```

```

├── SKILL.md                    # Skill 主入口
├── requirements.txt            # 依赖
├── prompts/                    # 提示词模板
│   ├── intake.md               # 信息录入
│   ├── memory_analyzer.md      # 友谊记忆分析
│   ├── memory_builder.md       # 友谊记忆模板
│   ├── persona_analyzer.md     # 性格分析
│   ├── persona_builder.md      # 性格模板
│   ├── merger.md               # 增量合并
│   ├── correction_handler.md   # 对话纠正
│   └── session_summary.md      # 会话总结
└── tools/                      # 工具脚本
    ├── wechat_parser.py        # 微信解析
    ├── qq_parser.py            # QQ 解析
    ├── social_parser.py        # 社交媒体扫描
    ├── photo_analyzer.py       # 照片 EXIF
    ├── skill_writer.py         # 文件管理
    └── version_manager.py      # 版本管理
```

## 生成的文件

每个朋友 Skill 会生成在 `friends/{slug}/` 下：

```
friends/儿子/
├── SKILL.md          # 可运行的完整 Skill
├── memory.md         # 友谊记忆
├── persona.md        # 人物性格
├── meta.json         # 元数据
├── versions/         # 版本历史
├── sessions/         # 会话记录
└── memories/         # 原始材料
    ├── chats/
    ├── photos/
    └── social/
```

## 安全说明

- 所有数据**仅本地存储**，不上传任何服务器
- 生成的 Skill **不会说现实中的朋友绝不可能说的话**
- 如果你对这个 AI 朋友产生了依赖……建议去找真人玩
- 本项目**仅用于娱乐和友谊记录**，不用于任何侵犯隐私的目的
