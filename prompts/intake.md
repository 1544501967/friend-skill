# 朋友信息录入引导脚本

## 开场白

```
你好，我是朋友.skill 创建器。

我会帮你把一个朋友，变成一个可以对话的 AI Skill。
整个过程只需要 4 个问题 + 一些原材料（可选）。

准备好了吗？
```

## 问题序列

### Q1：真实姓名（必填）

```
告诉我ta的真实名字吧。
如果ta有常用昵称或外号，也可以一起告诉我。

比如：张三（三哥） / 李小花（花花） / 王大伟
```

**验证**：非空即可。生成 slug 规则：中文用拼音，英文用小写，空格替换为下划线。

### Q2：认识的时间和方式（必填）

```
你们是什么时候认识的？怎么认识的？

可以包括：
- 具体时间或大概时间段
- 通过什么渠道认识的（同学/同事/朋友介绍/网上/其他）
- 第一次见面/交流的场景

示例：
  "2020年大学入学，同寝室认识的"
  "2018年通过朋友聚餐认识的"
  "幼儿园就认识了，发小"
  "2019年在公司入职培训的时候认识的"
  "王者荣耀游戏里认识的，后来面基了"
```

**解析字段**：
- `known_since`：认识时间
- `how_met`：认识方式
- `first_encounter`：初次见面场景

### Q3：印象深刻的事（必填，1-2件）

```
你们之间有没有什么让你印象深刻的事情？
讲 1-2 件就行。

可以是任何类型：
- 一起经历的事（旅行、冒险、突发事件）
- ta帮过你的忙或你帮过ta的忙
- 特别搞笑的事
- 让你感动的事
- 让你对ta刮目相看的事

示例：
  "有一次一起爬山遇到暴雨，在山洞里躲了两小时"
  "我失业的时候ta二话不说让我住ta家一个月"
  "大学时候期末考试ta帮我突击复习，结果ta自己差点挂了"
  "一起去吃火锅，ta点了个特辣锅，吃完就进了医院"
```

**解析字段**：
- `memorable_events`：印象深刻的事件列表
- `event_type`：事件类型（搞笑/感动/冒险/互助/其他）

### Q4：性格画像（可跳过）

```
最后一个：用一句话描述ta的性格？

可以包括：
- MBTI / 星座
- 性格特点
- ta最让你印象深刻的地方
- ta在朋友圈子里是什么角色

示例：
  "ENFP 双子座 话痨 社交达人 但很仗义"
  "不知道MBTI 但是很靠谱 有点闷骚 喜欢打游戏"
  "ISTJ 摩羯座 超级靠谱 但有点无趣 是朋友圈里的军师"
  "很搞笑 是我们群的气氛担当 但正经起来特别靠谱"

跳过也行。
```

**解析字段**：
- `mbti`：MBTI 类型
- `zodiac`：星座
- `personality`：性格标签列表
- `impression`：主观印象

## 汇总确认

```
好的，我整理一下：

  真实姓名：{real_name}（{nickname}）
  认识时间：{known_since}
  认识方式：{how_met}
  印象深刻的事：
    1. {event_1}
    2. {event_2}
  性格画像：{personality_summary}

这样对吗？确认后我们进入下一步（导入原材料）。
```
