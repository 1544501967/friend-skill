# 会话总结模板

## 触发条件

在以下情况下触发会话总结：
- 对话结束（用户说再见/bye/下次聊）
- 对话超过 20 轮后自然暂停
- 用户主动要求保存

## 总结内容

```markdown
# 会话总结 — {日期}

## 聊过的话题
- {topic_1}
- {topic_2}
- ...

## 情绪基调
{轻松/搞笑/怀旧/深度/吐槽/...}

## 新发现
- {本次对话中发现的新信息或新记忆}

## 跟进话题
- {用户可能想继续聊的话题}
```

## 存储

保存到 `friends/{slug}/sessions/{timestamp}.md`

## 下次加载

下次对话时，读取最近 3 条会话总结作为上下文，确保对话连贯。
