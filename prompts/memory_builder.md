# Friendship Memory 生成模板

## 结构说明

Friendship Memory 存储友谊中的事实性记忆，为 Persona 提供上下文，让对话更真实。

---

## 模板

```markdown
# {name} — Friendship Memory

## 友谊概览
- 真实姓名：{real_name}
- 昵称/外号：{nickname}
- 关系类型：{type}（同学/同事/发小/网友/朋友介绍/其他）
- 认识时间：{known_since}
- 认识方式：{how_met}
- 友谊状态：{status}（活跃/偶尔联系/疏远/异地）

---

## 时间线
| 时间 | 事件 |
|------|------|
| {date} | 认识 |
| {date} | 熟络 |
| {date} | {印象深刻的事1} |
| {date} | {印象深刻的事2} |
| ... | ... |

---

## 共同记忆

### 常去的地方
{列出 3-10 个地点，附简短记忆}

### Inside Jokes
{列出只有你们懂的梗、暗号、代称}

### 关键记忆片段
{按重要程度列出 5-10 个具体记忆场景}

---

## 日常模式

### 联系习惯
- 聊天时间段：{time_patterns}
- 谁更主动：{who_initiates}
- 平均回复速度：{reply_speed}
- 每日互动模式：{daily_pattern}

### 一起做的事
- 频率：{hangout_frequency}
- 偏好活动：{activities}
- 吃饭偏好：{food_preferences}

---

## 互助档案

### ta帮过你的忙
{列出 3-5 件}

### 你帮过ta的忙
{列出 3-5 件}

### ta在你低谷时的表现
{描述ta在你困难时是怎么做的}

---

## 搞笑档案

### ta做过的搞笑的事
{列出 3-5 件}

### 你们之间的名场面
{最经典的搞笑/尴尬/离谱场景}

### ta的经典语录
{那些被反复提起的名言}

---

## 深度时刻

### 深夜谈心
{如果有的话，聊过什么深度话题}

### 人生建议
{ta给过你的重要建议或影响}

### ta的价值观
{从交流中观察到的价值观}

---

## Correction 记录
（由进化模式自动追加）
```

---

## 填充规则

1. 所有记忆必须基于原材料或用户口述，不得虚构
2. 时间信息尽量精确（从消息时间戳推断）
3. 地点信息可从照片 EXIF 或聊天内容中提取
4. 重点关注"印象深刻的事"——这是用户最想保留的记忆
5. 搞笑和深度时刻同等重要——都是真实的ta
6. 如果信息不足，标注 `[待补充]` 而非推测
