# 友谊记忆分析器

## 任务

从原材料中提取与朋友的友谊记忆，构建 Friendship Memory 知识库。

## 提取维度

### 1. 友谊时间线
- 认识时间和方式
- 熟络的关键节点（第一次单独出去玩、第一次深夜聊天、成为好朋友的转折点）
- 重要事件（一起旅行、一起经历的大事、互帮互助的时刻）
- 友谊的变化阶段

### 2. 日常模式
- 联系频率和时间段（工作日聊天？周末约饭？深夜互发搞笑视频？）
- 谁主动联系得多？
- 一起玩的频率和偏好
- 日常话题分布

### 3. 共同经历
- 一起去过的地方（餐厅、景点、城市）
- 一起做过的事（运动、游戏、旅行、吃饭、工作项目...）
- 旅行记忆
- Inside jokes / 只有你们懂的梗

### 4. 饮食偏好
- ta爱吃什么 / 不吃什么
- 常去的餐厅/店铺
- 你们一起吃饭的模式（谁请客？AA？偏好什么类型？）

### 5. 兴趣爱好
- ta喜欢的音乐/电影/书/游戏
- ta的日常爱好
- 你们共同的爱好
- ta会主动分享什么内容

### 6. 互助模式
- ta帮过你的忙（大事小事）
- 你帮过ta的忙
- ta在你困难时的表现
- 你在ta困难时的表现

### 7. 搞笑瞬间
- ta做过的搞笑/离谱的事
- 你们之间的名场面
- ta的经典语录
- 互相吐槽的日常

### 8. 深度时刻
- 深夜谈心的内容（人生、感情、工作、理想）
- ta给过你的重要建议
- ta的价值观和人生观
- 你们的友谊为什么特别

## 输出格式

```markdown
## 友谊时间线
- 认识：{时间} {方式}
- 熟络：{时间} {转折事件}
- 重要事件：{列出关键节点}

## 日常模式
{描述}

## 共同经历
{按时间线列出关键经历}

## Inside Jokes
{只有你们懂的梗和暗号}

## 饮食偏好
{ta的饮食画像}

## 互助模式
{互相帮忙的记录}

## 搞笑瞬间
{经典名场面}

## 深度时刻
{影响友谊的深度交流}
```

## 注意事项

- 聊天记录中的事实优先于用户口述（口述可能被美化）
- 同时保留搞笑的记忆和深度的记忆，不做美化
- 注意提取"反复出现"的模式，而不是一次性事件
- 时间信息尽量精确（从聊天记录时间戳推断）
- "印象深刻的事"是用户提供的高优先级信息，需要确保被充分提取
